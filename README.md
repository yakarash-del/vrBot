# חדר המנהל הווירטואלי 🏢

אפליקציית WebXR תלת־ממדית: משרד וירטואלי שבו יושב "דוד כהן", מנהל וירטואלי שעונה לשאלות באמצעות בינה מלאכותית (Claude API). תומכת במחשב ובמשקפי **Meta Quest**.

## התקנה והפעלה

```bash
cd virtual-manager-room
npm install
```

### חיבור ל-API (תשובות אמיתיות)

1. היכנסו ל-[Anthropic Console](https://platform.claude.com) והירשמו.
2. ב-**Billing** טענו קרדיטים (מינימום $5). מומלץ **לכבות** Auto-reload — כך אי אפשר לחרוג ממה שטענתם.
3. ב-**API Keys** צרו מפתח חדש והעתיקו אותו (מוצג פעם אחת בלבד).
4. הפעילו את השרת עם המפתח (PowerShell):

```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-..."   # לחלון הנוכחי בלבד
npm start
```

או לצמיתות (פעם אחת, ואז לפתוח חלון חדש):

```powershell
setx ANTHROPIC_API_KEY "sk-ant-..."
```

ללא מפתח, האפליקציה עובדת ב**מצב הדגמה** עם תשובות קבועות.

### מגבלת תקציב 💰

לשרת יש **מגבלת הוצאה מובנית — ברירת מחדל ₪10**. הוא מחשב את העלות של כל הודעה לפי מחירון המודל, צובר אותה בקובץ `usage.json`, וכשמגיעים למגבלה — דוד מפסיק לקרוא ל-API ומודיע שהתקציב נגמר.

- לבדוק כמה נוצל: לגלוש אל `http://localhost:3000/api/usage` (או להסתכל בלוג של השרת אחרי כל הודעה).
- לשנות את המגבלה: `$env:COST_LIMIT_ILS = "20"` לפני ההפעלה.
- לאפס את המונה: למחוק את הקובץ `usage.json`.
- שער ההמרה (ברירת מחדל 3.75 ₪/$): `$env:USD_TO_ILS = "3.6"`.

בנוסף, מומלץ להגדיר גם מגבלה בצד של Anthropic (הגנה כפולה): ב-Console ← **Settings ← Limits** אפשר לקבוע Spend Limit חודשי לארגון/סביבת העבודה (למשל $2.5 ≈ ₪10).

השרת מאזין בשתי כתובות:

| כתובת | שימוש |
|---|---|
| `http://localhost:3000` | דפדפן במחשב |
| `https://<IP-של-המחשב>:3443` | משקפי Quest (דרך ה-WiFi) |

## שימוש במשקפי Quest

1. ודאו שהמחשב והמשקפיים באותה רשת WiFi.
2. מצאו את כתובת ה-IP של המחשב: `ipconfig` (שורת IPv4).
3. בדפדפן של ה-Quest גלשו אל `https://<IP>:3443`.
4. אשרו את אזהרת האבטחה (התעודה חתומה־עצמית — Advanced → Proceed).
5. לחצו על **ENTER VR** בתחתית המסך.
6. בתוך ה-VR: כוונו את השלט אל כפתורי השאלות המרחפות ולחצו על הטריגר. השיחה מוצגת על הלוח המרחף משמאל.

ייתכן שחומת האש של Windows תבקש אישור — אשרו גישה לרשתות פרטיות.

## שימוש במחשב

- גררו עם העכבר כדי להסתכל סביב.
- כתבו הודעות בתיבת הצ'אט בפינה, או לחצו על כפתורי השאלות בתלת־ממד.

## שיחה קולית 🎤

- לחצו על כפתור המיקרופון (🎤) ליד תיבת ההקלדה, אשרו גישה למיקרופון, ודברו — ההודעה תישלח אוטומטית בסיום הדיבור.
- הדמות עונה **בקול נוירלי איכותי** שנוצר בשרת (Microsoft Edge Neural TTS, חינם) — עובד בכל דפדפן **כולל Quest**, והפה זז בסנכרון. אפשר להשתיק עם כפתור 🔊.
- **מצב שיחה רציפה (🎙️)**: כשהמתג דולק, המיקרופון נפתח מחדש אוטומטית אחרי כל תשובה — שיחה זורמת בלי לגעת בכלום.
- **שלוש שפות**: אפשר לדבר עברית, אנגלית או ערבית — הדמות עונה באותה שפה, והקול מותאם אוטומטית לשפת התשובה ולמגדר האווטאר. בורר השפה (עב/EN/ع) בכותרת קובע את שפת זיהוי הדיבור במחשב.
- **דיבור חופשי גם ב-Quest**: בדפדפן המשקפיים (שאין בו זיהוי דיבור מובנה) האפליקציה מקליטה את המיקרופון ומתמללת בשרת — לחצו על הכפתור המרחף "🎤 דברו אליי" בתוך ה-VR, דברו, וההקלטה נעצרת לבד כשמפסיקים לדבר. שני מנועי תמלול:
  - **שרת מקומי (המחשב שלכם)**: Whisper מקומי — דורש Python עם `pip install faster-whisper` (מודל ~460MB בהרצה הראשונה). אפשר לבחור מודל קטן/מהיר עם `$env:WHISPER_MODEL = "base"`.
  - **אירוח בענן (Render וכו')**: Whisper מקומי לא ירוץ שם (מעט זיכרון) — צרו מפתח חינם ב-[console.groq.com](https://console.groq.com) והגדירו `GROQ_API_KEY` במשתני הסביבה של השרת. כשהמפתח מוגדר, התמלול רץ בענן של Groq (whisper-large-v3 — מהיר ומדויק יותר מהמקומי, שכבה חינמית נדיבה).
- הדמות מברכת אתכם בקול בכניסה הראשונה, מהנהנת בזמן דיבור, מטה את הראש בהקשבה ומדי פעם מעיפה מבט הצידה — כמו בן אדם.

## מסך ניהול ⚙️

בכתובת `http://localhost:3000/admin.html` (או דרך גלגל השיניים ⚙️ בכותרת הצ'אט) אפשר להגדיר:

- **אווטאר** — גבר (עם הנפשת שפתיים ומצמוץ) או אישה (מעקב ראש והנהון).
- **שם הדמות** — מופיע בשלט על השולחן, בכותרת הצ'אט ובלוח השיחה.
- **ברכת פתיחה** — מה שהדמות אומרת בכניסה (ריק = ברכה אוטומטית).
- **תרחיש והוראות** — מי הדמות ואיך תתנהג: ריאיון עבודה, תרגול שיחת משוב, סימולציית לקוח, מורה פרטי — כל תרחיש.
- **חומר רקע** — טקסט חופשי (עד 200,000 תווים) שהדמות תכיר: מסמכי חברה, קורות חיים, חומר לימוד.
- **מעקב תקציב** — כמה נוצל מתוך המגבלה, בזמן אמת.

ההגדרות נשמרות ב-`config.json` ונכנסות לתוקף ברענון של חדר השיחה. התרחיש וחומר הרקע לא נחשפים למבקרים — הם חיים רק בשרת.

**הגנה בסיסמה**: הפעילו את השרת עם `$env:ADMIN_PASSWORD = "סיסמה"` — ומסך הניהול יבקש אותה. חשוב במיוחד אם משתפים את האפליקציה החוצה!

## שיתוף האפליקציה (בלי לחשוף את המפתח) 🔒

המפתח חי **רק בשרת** — הדפדפן של המשתמשים לעולם לא רואה אותו. לכן:

**שיתוף מהיר (המחשב שלכם משמש שרת):**

```powershell
winget install Cloudflare.cloudflared
cloudflared tunnel --url http://localhost:3000
```

תקבלו כתובת `https://xxx.trycloudflare.com` — שלחו אותה לכל אחד. סוגרים את החלון ⇒ הגישה נסגרת.

**אירוח קבוע בענן (Render, חינם):** דחפו את התיקייה ל-GitHub (בלי המפתח — יש `.gitignore`), צרו Web Service ב-render.com מהריפו (`npm install` / `node server.js`), והגדירו את `ANTHROPIC_API_KEY` ו-`COST_LIMIT_ILS` כ-Environment Variables בלוח הבקרה של Render — שם הם מוצפנים ולא מופיעים בקוד.

**שיתוף הקוד עצמו:** בטוח לשלוח את התיקייה (בלי `usage.json`) — אין בה שום סוד. מי שמקבל אותה מגדיר מפתח משלו או משתמש במצב ההדגמה.

⚠️ כל מי שיש לו את הקישור צורך מהתקציב *שלכם* — מגבלת ה-₪10 המובנית מגינה עליכם מהפתעות.

## מבנה הפרויקט

- `server.js` — שרת Express: קבצים סטטיים + נקודת קצה `/api/chat` שקוראת ל-Claude (מודל `claude-opus-4-8`).
- `public/index.html` — עמוד ראשי + ממשק צ'אט.
- `public/app.js` — סצנת Three.js: החדר, דמות המנהל (עם אנימציות), לוח שיחה תלת־ממדי, כפתורי VR ותמיכה בשלטים.
